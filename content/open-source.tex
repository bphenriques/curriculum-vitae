    {\href{https://github.com/bphenriques/selfhost-nix}{bphenriques/selfhost-nix}}
    {January 2025 - Present}
    {
    \begin{openSourceItems}
        \item \textbf{Reusable NixOS framework} that removes the repetitive setup behind self-hosting: one declaration wires reverse proxy, single sign-on, monitoring, backups, and notifications --- with swappable implementations.

        \item \textbf{Shipped as a real project:} VM integration tests in CI, a documentation site, and a starter template.
    \end{openSourceItems}
